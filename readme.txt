Automatically fills in the Alt text, Caption, and Description from the image file name of new image uploads.

== Description ==

When you upload a new image in WordPress, this plugin will use the file name to automatically fill in the Title, Alt text, Caption fields, and Description so you don't have to do it manually.

== Installation ==

Activate the plugin through the 'Plugins' menu in WordPress.
